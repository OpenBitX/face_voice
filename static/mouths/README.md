# Rhubarb Lip-Sync 嘴型素材

源视频：`docs/videos/制作说话的眼睛嘴巴视频.mp4`
（1280×720 / 24fps / 8s / H264）

## 统一处理参数

- 裁剪 + 缩放：`crop=900:450:190:200,scale=500:250`
  - 原图 1280×720 中抠出 x∈[190,1090], y∈[200,650] 的 900×450 区域
  - 再统一缩放到 **500×250**（保持宽高比）
  - 起点 y=200 保留充足的下巴空间，高度 450（底边到 y=650）确保下巴完整
- 所有 9 张 PNG 尺寸一致：**500×250**，透明背景（PNG alpha 通道）
- 一键重新生成：`uv run python scripts/gen_mouths.py`

### 抠图算法说明

BFS 泛洪填充**只从顶边 + 左右列播种，跳过底边**：
- 跳过底边原因：底边靠近下巴皮肤过渡区，若从底边播种会把下巴误识别为背景删除
- 底部黑色背景经由左/右侧连通黑区自然覆盖，不需要单独从底边播种

## 单张复刻（如需重跑某张）

```bash
ffmpeg -y -ss 1.00 -i "docs/videos/制作说话的眼睛嘴巴视频.mp4" -vframes 1 \
  -vf "crop=900:450:190:200,scale=500:250" \
  "static/mouths/shape-D.png"
```

所有时间戳对应的 `-ss` 值（秒）：
A=2.00, B=0.25, C=2.50, D=1.00, E=4.00, F=5.50, G=5.75, H=0.75, X=2.00

## 微调提示

- 若嘴位置整体偏上/偏下，调整 crop 的 Y（当前 200）。
- 若下巴仍被截断，增大 crop 的 H（当前 450，最大不超过 720-Y）。
- 若左右留白不均，调整 crop 的 X（当前 190）。
- 若最终帧分辨率想变更，改 `scale=W:H`（保持宽高比）。
